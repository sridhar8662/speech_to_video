# 🎥 Text to Video Generator

A simple and interactive **Text-to-Video Generator** built with **Streamlit**. Users can enter a scene description and view a sample generated video. This project demonstrates the basic workflow of a text-to-video application and can be extended with AI video generation APIs such as RunwayML, Pika, or Stability AI.

## 🚀 Features

* Clean and user-friendly Streamlit interface
* Text prompt input for scene descriptions
* One-click video generation simulation
* Embedded sample video playback
* Success and warning notifications
* Easy to customize and extend

## 🛠️ Technologies Used

* Python
* Streamlit

## ▶️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/text-to-video-generator.git
```

2. Navigate to the project directory:

```bash
cd text-to-video-generator
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app.py
```

## 🎬 How It Works

1. Enter a text prompt describing a scene.
2. Click the **Generate Video** button.
3. The application displays a sample video as a placeholder.
4. Replace the sample video logic with an AI video generation API for real video creation.

## 📸 Example Prompt

```text
A person playing with a ball in a sunny park
```
